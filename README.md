# The Python Cookbook — From Zero to Job-Ready

An interactive, single-file Python learning cookbook that takes you from absolute beginner to job-ready across automation, data analytics, and AI/ML.

**Live site:** https://chezhian0599.github.io/python_cookbook/

## What's inside

- **55 modules** across 10 parts — from "what's a variable?" to shipping an open-source library and landing your first Python job
- Built around **memory-science principles**: one idea per module, everyday analogies, predict-before-reveal, and recall checkpoints
- Every module follows the same structure: goal → analogy → plain-English concept → annotated code → predict-the-output → three real-world examples (automation / analytics / AI) → beginner traps → graded practice problems with solutions → recall quiz → TL;DR → 30-second interview answer
- **Progress tracking** saved to your browser (localStorage) — persists across sessions
- **Auto-built flashcards** from modules you've completed
- **Searchable glossary** aggregated from every module
- **Dark mode** + **print-to-cheat-sheet** view
- **Mobile-friendly** responsive layout
- **Zero dependencies** — one self-contained HTML file, works fully offline

## How to use it

Just open `index.html` in any modern browser. Or host it for free — see below.

## Hosting it yourself

The easiest way is GitHub Pages (free):

1. Fork or create a new public repo
2. Upload `index.html` (rename from `python-cookbook.html` if needed)
3. **Settings → Pages → Source: Deploy from branch → main → / (root) → Save**
4. Wait ~30 seconds, visit `https://your-username.github.io/repo-name/`

Your progress is stored per-browser via localStorage, so different devices have independent progress — no backend needed.
